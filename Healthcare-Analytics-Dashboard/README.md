# Healthcare Analytics Dashboard

## Project Overview
This project analyzes 55,500+ healthcare records using SQL and Power BI.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel

## Dataset
The dataset contains healthcare patient records including:
- Patient Demographics
- Medical Conditions
- Hospitals
- Doctors
- Insurance Providers
- Billing Information

## Analysis Performed
- Total Patients Analysis
- Gender Distribution
- Average Age Analysis
- Medical Condition Analysis
- Hospital Performance Analysis
- Doctor Performance Analysis
- Billing Analysis
- Insurance Provider Analysis
- Length of Stay Analysis

## Key Insights
- Total Patients: 55,500
- Average Age: 51.54 Years
- Average Length of Stay: 15.51 Days
- Identified top hospitals and doctors by patient count
- Analyzed revenue generated by insurance providers
