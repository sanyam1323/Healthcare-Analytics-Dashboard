# Healthcare-Analytics-Dashboard
## Project Overview

This project analyzes 55,500+ healthcare records using SQL and Power BI. The goal was to perform data cleaning, exploratory data analysis, and create an interactive dashboard to uncover insights related to patients, hospitals, doctors, medical conditions, insurance providers, and billing trends.

## Tools Used

* MySQL
* SQL
* Power BI
* Excel

## Dataset Information

* Total Records: 55,500+
* Domain: Healthcare
* Dataset Type: Patient Healthcare Records

## Data Cleaning

* Checked for missing values
* Verified data types
* Renamed columns for consistency
* Reviewed dataset structure
* Validated records before analysis

## SQL Analysis

The following analyses were performed:

* Total Patients
* Male vs Female Patients
* Average Patient Age
* Medical Condition Distribution
* Medical Conditions by Gender
* Top Hospitals by Patient Count
* Top Doctors by Patient Count
* Average Length of Stay
* Longest Patient Stays
* Average Billing Amount
* Revenue by Medical Condition
* Revenue by Insurance Provider
* Insurance Provider Performance

## Dashboard Features

* Total Patients KPI
* Average Age KPI
* Average Billing Amount KPI
* Average Length of Stay KPI
* Medical Condition Analysis
* Hospital Performance Analysis
* Doctor Performance Analysis
* Insurance Provider Analysis
* Revenue Analysis

## Key Insights

* Analyzed 55,500+ healthcare records.
* Average patient age: 51.54 years.
* Average hospital stay: 15.51 days.
* Identified top-performing hospitals and doctors.
* Analyzed revenue contribution by insurance providers.
* Compared medical conditions across patient groups.
